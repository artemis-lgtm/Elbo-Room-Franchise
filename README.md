# Elbo Room Heritage License Program

**[View the full strategy site](https://artemis-lgtm.github.io/Elbo-Room-Franchise/)**

## Strategic Expansion & Property Acquisition

---

### Vision

Transform the Elbo Room — Fort Lauderdale's 88-year-old landmark bar — from a single iconic location into a national heritage bar brand through a licensing program that funds property acquisition by 2030-2032.

**Positioning:** "The bar your neighborhood can't afford to lose." Not a franchise system. Not a themed chain. A heritage licensing model that helps bars become institutions.

### The Problem

- Lease expires 2036
- Current borrowing capacity: ~$8M
- Target purchase price: ~$15M minimum
- Gap: ~$7M

### The Solution

The Heritage License Program generates revenue to close the gap while building a brand that makes the Fort Lauderdale flagship more valuable, not less.
